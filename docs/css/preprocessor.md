# CSS 预处理器

## 概述

* 基于 CSS 的另外一种语言
* 通过工具编译成 CSS
* 添加了很多 CSS 不具备的特性
* 能提升 CSS 组织方式

## 特性

* 嵌套和反映层级关系
* 变量和计算，减少重复代码
* extend 和 mixin 代码片段
* 循环，适用于负责有规律的样式
* import CSS 文件模块化

## 常用 CSS 预处理器

* less
* sass
* stylus

## 预处理器框架

* Sass-Compass
* Less-Lesshet / EST
* 提供现成的 Mixin
* 类似 JavaScript 类库，封装常用功能
